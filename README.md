This is my version of a Todo App for The Odin Project.
- Saves in local storage per the curriculum
- Create 'Projects' to store lists of Todos for specific categories. i.e. 'Home', 'Work', etc.
-   Within Projects, bulk delete, edit, duplicate, and mark complete todos (Work in progress)
- Create 'Todos' within projects. i.e. 'Mow the lawn', 'Do the dishes', etc.
-   Within Todos, store notes or add to a checklist to mark off items (Work in progress)
-   Dark/Light mode toggle that also saves theme state in local storage.
